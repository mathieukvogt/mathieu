## Installation
To install this typeface, follow these steps:
1. Download the font files from the repository.
2. Install the font files on your system:
   - **Windows**: Right-click the font file and select 'Install'.
   - **Mac**: Double-click the font file and select 'Install Font'.
   - **Linux**: Copy the font files to your `~/.fonts` directory.
